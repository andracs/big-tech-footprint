# Visualisering af techgiganternes CO2-udladninger

Denne app viser faktiske og forventede CO2e-udledninger for Microsoft, Google, Apple, Amazon og Meta/Facebook. Den bruger en interaktiv graf til at sammenligne rapporterede Scope 1+2+3-tal, målbaner og trendbaserede forventninger mellem 2020 og 2026.
